# Basic Exercises
This folder contains some basic exercises to learn how to write code in python language.

## Exercises
### Exercise 1:
Write a script that accepts an input consisting of an integer representing a radius.
- Calculates the area of ​​the corresponding circle, and prints the result.

### Exercise 2:
Write a script that accepts three integers.
- If they are all different, it prints the sum of the three;
- If two are equal, it prints the sum of the equal numbers divided by the different one;
- If all three are equal, it prints the result of (n + n)^n

### Exercise 3:
Write a script that accepts two integers.
- If both values, their sum, or their difference is 5, it prints "True"
- otherwise it prints "False"
- If the inputs entered are nonnumeric, it prints "The inputs must be integers!"
